Using Vizi

Last updated: 1st Oct, 2017

OK, lets get going!

Here we will talk through the features in Vizi, how to use them and how to get the most of them. Remember, there are tooltips on each page that might be able to help as well.

User Interface

Logging in

Log into your account at 'vizi.school' or click the login button on the 'vizi.education'. Select the 'Remember me' checkbox to speed things in the future.

Sidebar

The sidebar runs down the right of Vizi and allows you to quickly navigate between features. Your sidebar will update as you are added to groups and collections so you have access to more data.

NavBar

The navbar appears at the top of a page when you have more options of a page. For example, if you are on a group page you can view your markbook, dashboard and students. Keep an eye on the navbar to fully explore features in Vizi.

Links

Nearly everything in Vizi is a link. If you think you should be able to click on it, you probably can. This allows you to move between pages and find information as quickly and intuatively as possible.

Markbook

My Markbook

Sidebar (Markbook)

What is your markbook?

A markbook is an aggregation of assessments for each of a user's groups. The My Markbook page shows all of a users markbooks organised on seperate display cards. The cards show key metrics relating to each markbook. This includes the number of students, the total assessments set for that group and the number of incomplete assessments for that group. Summary metrics are shown at the top of the page that relate to all markbooks for a user.

Using your markbook?

Each metric can be selected and the detail behind the metric viewed. Clicking the arrow next to the group name will navigate a user to the Markbook - Grid View, which shows the indivudal assessments contributing to that markbook. Cards can also be rearranged by unlocking the Markbook screen and dragging the cards around the page. Users can save a layout by clicking the 'save layout' button and the My Markbook page will reload in that format on the next visit. Users can view all of their assessments in tabular format by clicking on the Assessments link on the navbar.

My markbook - Assessments

Sidebar (Markbook) Navbar (Assessments)

What are your assessments?

All assessments ever created by a user are displayed in tabular format. This allows the user to easily search and edit assessments across all of their markbooks.

Using your assessments?

Users can search, order and edit their assessments from the assessment table. To edit an assessment, select an assessment and click the 'edit' button.

Markbook - Grid view

Sidebar (Groups) Navbar (Markbook)

What is your markbook grid view?

Once a group has been selected from the groups dropdown in the sidebar, a user can select 'Markbook' on the navbar and view all grades for the selected group. These grades are displayed in a conventional markbook grid view. The markbook shows the the assessment title, set and due date and grade type across the x axis and the students down the y axis. This will show every assessment for that group across the full academic year.

Using your assessments?

The markbook can be searched and filtered by student name using the search box in the top right of the page and the and the filter arrows in the student name columns.

Colour can be added to the markbook based on different parameters by selecting the 'Colour markbook' button. A user can select to colour the markbook by 'Grade' or by 'incomplete assessments', this will add a colour scale between green and red to every completed grade.

New assessments can be created for the group by selecting 'Add assessment' button and old assessments can be updated by selecting the assessment name in the column header. When the assessment name is selected the user will get the option to enter new marks, edit assessment details and show/hide comments for that assessment.

Add an assessment

Sidebar (Groups) Navbar (markbook) Button (Add assessment)

What are your assessments?

A user can select the 'Add assessments' button and create new assessments and begin to record grades for a selected group. Users can create custom names for the assessment or reuse a previously used assessment name (this will allow marks to be aggregated in the dashboard view).

Using your assessments

When creating a new assessment the subject, assessment type and grade type(s) must be defined. These can be selected from the subjects, assessment types and grade types created using the admin tools area of Vizi (new subjects, assessment types and grade types can be added to Vizi by teachers with the appropriate admin permissions). By selecting different grade types the user will be required to fill in grades for each student for all selected grade types for the assessment to be completed.

Set date, due date and topic(s) can also be optionally selected for the assessment. The dates will dictate notifications on a users timeline and reminders to collect assessments. Topics can be assigned to assessments by searching for previously used topics or creating new topics using the 'Can't find what you're looking for?' link. Topics are used to manage and filter performance data.

Once the required information has been entered into the assessment details, the user can create the assessment and begin to enter marks for each student in the group.

Using your assessments

Users can edit previously created assessments by selecting an assessment title whilst in a markbook and clicking the 'edit assessment details' button. **A user cannot remove grade type from an assessment if grades have already been recorded for that grade type. To remove a grade type all recorded grades for that grade type must be removed when editing grades for that assessment.**.

Assessment marks

Sidebar (Groups) Navbar (Markbook) Modal on assessment header

What are assessment marks?

Users can enter and modify marks once an assessment has been created by selecting the title of an assessment whilst in a markbook and clicking the 'Enter grades' button.

Using assessment marks?

Users can enter grades, statuses and comments for each student from the dropdown lists or by manually entering the grade or status (this is restricted to the grades and statuses created in admin tools). Once grades have been entered or altered a user can click the 'Update Marks' button to save the grades or 'Cancel' to disregard the marks and go back to the previous screen. The quick fill function allows a user to enter consistent marks for all students.

Dashboards

What are dashboards?

Dashboards are at the heart of Vizi. They enable the user to display, interogate and draw conclusions from performance data. Dashbaords can be viewed by collection, group and user and can display multiple measures of performance simultaneously.

Users can create dashboards simply by clicking on the dashboard button in the navbar menu when on collection, group or user profiles. If a user viewing that dashboard for the first time, Vizi will use it's autobuild function to build a complete dashboard using the data available. The autobuild function uses data visualisation best practice, whilst considering the amount and type of data available to build a dashboard that can be used and create insight immediately.

Creating and saving dashboards

Users can create, save and load their own dashboards. Once a dashboard is loaded, the user can unlock the dashboard by selecting the unlock icon at the top right of the dashboard. This unlocks all individual elements on the dashboard and allows the user to move, resize and remove the elements to create custom layouts.

Rearranging dashboards Resizing of elements is controlled by dragging the double headed arrow icon in the bottom right of the element until it reaches the desired size. Moving elements is controlled by dragging from anywhere in the element and dropping in a new location. Vizi will rearrange other elements to make room for users moving elements and then new location for the element is indicated by a dashed border that appears on the dashboard. Once the desired layout has been acheived a user can click the lock icon to lock the dashboard and go back to the normal dashboard functionality.

Add elements Users can add new elements to a dashboard by clicking the cog icon and in the top right of the dashboard and selecting the 'Add chart' option. This will open a modal and give the user a drop down list of metrics can display on the new chart. Once a metric has been selected either the 'Create chart' button will become active or if the metric can be displayed in multiple formats there will be a second option to select the chart type. When a new chart is created it will appear at the bottom of the dashboard.

Save as layout Custom layouts can be saved by clicking the cog icon and selecting 'Save layout as...'. If a previously saved layout is loaded, users can select the 'Save Layout' option to update the layout with any changes that have been made.

Load layout Previously saved layouts can be loaded by clicking the cog icon and selecting 'Load Layout'. Users can choose from their previously saved dashboards.

A custom dashboard can be set as the default dashboard for a collection, group or user by clicking the cog icon and selecting 'Set Default'. When that collection, group or user is revisited, Vizi will load the default layout. Users can revert back an auto-built dashboard at anytime by selecting the 'Auto-Build' option.

Extracting data

When using a dashboard in Vizi, a user is always able to view and extract the raw data that makes up the dashboard. Below the dashboard there is a data table that displays a users data in tabular format. When a user filters their dashboard or adds and removes data the data table will update reflect exactly what is seen in that dashboard

To extract the data the user can select from the several data formats listed at the top of the data table. By clicking the desired data format Vizi will automatically download the data onto a users device. The data can also be copied for use in other applications by clicking the 'Copy' button.

Calendars

My Calendar

Sidebar (My Calendar)

What is your calendar?

Users can view a calendar showing all events and tasks for groups associated to a user.

Using your calendar?

Users can filter and edit calendar colours using the group cards to the right of the calendar. These changes can be saved for future use by clicking the 'Save Changes' button.

Users can select month, week or day to change the view of the calendar.

Tasks

My Tasks

Sidebar (My Tasks)

What are tasks?

The My Tasks page is used to manage a user's upcoming and completed tasks. Tasks can be created on an adhoc basis to manage to do lists and can be auto-created upon setting an assessment (unless the 'Create Task' tick box is unticked) helping users to stay on top of their marking.

Using Tasks?

The 'My Tasks' page displays a users incomplete and complete tasks. It shows the incomplete tasks as a list at the top of the page and completed task as a list below. Users can mark tasks as completed by selecting the checkbox next to a task and they can mark tasks as incomplete again by selected the checkbox next to the completed tasks.

Users can add extra details to tasks by selecting the task title. This includes the following details:

  • Task description - A short description to add more detail to a task title.
  • The due date for that task - This will prompt a reminder to appear on the user's timeline.
  • Start and end time - If the task will take place over a discrete period of time a user can detail the exact time the task needs to be completed.
  • Groups - Tasks might relate to a particular groups or groups, in which case a user can add a group or mulitple groups to the task.
  • Task colour - Tasks will appear on a users calendar if the date parameters are set. Users can use the default colour for tasks, inherit the colour of a group or choose a custom colour.

Homescreens

Your homescreen

Sidebar (Home)

What is your homescreen?

A users homescreen is the hub of Vizi. It is what appears when a user first logs into Vizi and can be accessed by selecting 'Home' on the sidebar. The homescreen shows a users timeline, calendar, tasks and assessment details.

Timeline - The timeline displays key feedback relating to assessments and performance data relating to groups associated with a user. It also users to send and receive messages with other selected users. Timeline messages relating to student peformance can be explored by clicking the find out more button. This will generate a popup showing the data that has contributed to the message.

Calendar - The calendar shows a user's events by day, week or month. Using the options tab, a user can change the display to show certain event types to get a more refined view.

Tasks The tasks window shows a summary of outstanding tasks for that user. It allows users to mark tasks as completed by selecting the checkbox next to the task and add new tasks by clicking the 'Add task' button, entering the title of the new task and clicking 'Add'.

Upcoming assessments The upcoming assessments windows shows a user's upcoming assessments that are due for collection or for marking.

Assessment completion The assessment completion window shows the all outstanding assessments and shows the completion status as a percentage. This is based on the proportion of students that have a grade for the assessment.

Group Homescreen

Sidebar (Groups)(Group)

What is a group homescreens?

The group home view is configured in the same way as the user homepage but will only show data relating to the selected group. Users can use the navbar to move to the markbook, the performance dashboard and the student layout for that group.

Student layout

Sidebar (Groups)(Group) Navbar (Students)

What are student layouts?

A user can view, arrange and access details for students in a group.

Using student layout?

The student layout can be used to simply view students in a group and explore their details or to represent a physical classroom layout. Users can unlock the student layout by selected the unlock icon and rearrange the student cards to suit a classroom layout. Students can be clicked on to access the student's homepage and view information for that student.

User Homsecreen

Sidebar (Groups)(Group) Navbar (Student Layout)(Student icon)

What are is a user homescreen?

User home screens show details for that user. It will show generic user information such as name, DoB and the account creation date. It will also show the groups that the user is in, a calendar for that user displayed all events relating to the user and contact information.

User homescreens can also show bespoke personal details (see user fields) such as medical information or social security details. To enter and manage user information select a user in the 'Admin Tools', 'Users' (Users) and click the 'Edit' button.

Search box

What are is the search box?

The search box is located in the sidebar and allows a user to search and quickly access information in Vizi.
