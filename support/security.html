Data Security


Data security is a priority at Vizi. Vizi's data is stored in the cloud, designed to ensure high performance and resilience. We partner with Amazon Web Services (AWS) - the world's leading cloud provider, to globally offer a secure, high-performance, load-balanced architecture.

In the event of a security breach (a breach of security leading to the accidental or unlawful destruction, loss, alteration, unauthorised disclosure of, or access to, personal data. This includes breaches that are the result of both accidental and deliberate causes) Vizi will notify all clients immediately of any such breach and work with the client and AWS to resolve the breach as quickly as possible.

Vizi uses end-to-end HTTPS and IP restrictions to ensure your data is always encrypted and only accessible to authorised users.
